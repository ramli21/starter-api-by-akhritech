# starter-api-by-akhritech 🚀

A modern, robust, and production-ready Laravel 13 REST API Starter Kit integrated with TailwindCSS and Laravel Sanctum. This boilerplate is designed to slash your development time, giving you a rock-solid foundation with a pre-built Super Admin Portal and ready-to-use authentication APIs.

## ✨ Features (Free Tier)

### 🖥️ Super Admin Portal

- **Secure Login** – Dedicated, secure authentication for administrative access.
- **User Management** – Easily view, manage, and control user accounts.
- **Log Viewer** – Beautiful, interactive system log monitoring powered by `opcodesio/log-viewer`.

### 🔐 API Ready (Sanctum Secured)

- **Complete Auth Flow** – Pre-configured endpoints for Login and Registration.
- **Account Security** – Built-in Reset Password and Change Password endpoints.
- **Verification** – Native Email Verification API handling.

---

## 🛠️ Tech Stack

- **Framework:** Laravel 13
- **Styling:** TailwindCSS
- **API Authentication:** Laravel Sanctum
- **Monitoring:** opcodesio/log-viewer
